# elwoodberry



## Installation
Get Updates
```
$ sudo apt-get updates
```
Clone Github
```
$ git clone https://github.com/elwoodberry/elwoodberry.git
```
Change directory to the master branch
```
$ cd elwoodberry
```
Install packages  
```
$ sudo npm install
```
Run the application(in the background)
```
$ sudo nohup nodejs app.js &
```
Remove Git Project
```
$ sudo rm -rf elwoodberry tmp
```
